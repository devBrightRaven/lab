# 栞 Shiori

Static interactive reading map for 17 predictive-processing, judgment-quality, and human-AI decision-making texts.

## Decisions Resolved

- Detail panels include paper-specific work implications. When the source map does not explicitly state one, the panel labels it as `延伸推論`.
- Three layout modes are implemented: constellation, three-axis stack, and recommended sequence.
- Cross-axis links are SVG lines in constellation mode and accessible relation chips in every mode.
- Local markdown sources are shown as local paths with a copy button instead of `file://` links.

## Stack

Pure HTML, CSS, and vanilla JavaScript. No framework and no build step.
